# KiCad_Design_Guide

### 介绍
Kerman 同学的 KiCad 教程大作，集成电路测试 LoadBoard 设计指南。
这是一本高质量的 KiCad 入门手册，从设计 IC 测试载板的角度，系统地介绍了如何使用 KiCad 建库、设计原理图、到 PCB Layout。本书不只是 KiCad 的功能介绍，还加入了大量的工程思考，非常值得一读！
